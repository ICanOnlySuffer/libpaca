# Pocha's Game Engine

A simple game engine built on top of [SDL](https://libsdl.org).

Compiles for GNU+Linux and Mingw (from withing GNU+Linux).

## Installation

**dependencies:**

- [SDL](https://libsdl.org/download-2.0.php) (make)
- [pul](https://gitlab.com/ICanOnlySuffer/pul) (make)

**commands:**

	# make install clean

## Examples
