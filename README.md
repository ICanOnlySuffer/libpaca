# Pocha's Game Engine

A simple game engine built on top of [SDL](https://libsdl.org).

Compiles for GNU+Linux and Mingw (from withing GNU+Linux).

## Installation

**requirements:**

- [SDL](https://libsdl.org/download-2.0.php)
- [pul](https://gitlab.com/ICanOnlySuffer/pul)

**commands:**

	# make install clean
